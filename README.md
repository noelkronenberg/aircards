# AirCards

Simple URL-based interactions via AirDrop.  
Each interaction is saved and transferred through URL parameters.

## Set-up

1. install requirements:
    - Python (tested: 3.9.10)
    - Flask (tested: 2.1.3)
2. run ```python3 app.py```
3. open http://127.0.0.1:5000/
